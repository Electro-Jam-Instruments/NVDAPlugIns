# PowerPoint Comments - NVDA Addon

Accessible navigation for PowerPoint comments using NVDA screen reader.

## Status

**Version:** 0.0.1-beta (Phase 1)
**Status:** Early Development

## Current Features (Phase 1)

- Connects to PowerPoint via COM automation
- Detects current view type
- Auto-switches to Normal view (required for comment access)
- Extensive logging for debugging

## Installation

1. Download the `.nvda-addon` file from [Releases](https://github.com/Electro-Jam-Instruments/NVDAPlugIns/releases)
2. Double-click to install
3. Restart NVDA when prompted

## Requirements

- NVDA 2023.1 or later
- Microsoft PowerPoint 365 or 2019+
- Windows 10/11

## Debugging

View addon logs: NVDA menu > Tools > View Log (or press NVDA+F1)

Look for messages starting with "PowerPoint Comments addon"
