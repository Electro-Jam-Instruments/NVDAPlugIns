Fresh start for v0.1.0 (no beta history).

**Content:**
```markdown
# Changelog

## v0.1.0 (January 2026) - Initial Release

First stable release of PowerPoint Comments NVDA addon.

### Features
- Automatic "Has X comments" announcement on slide change
- "has notes" prefix for slides with meeting notes (marked with ****)
- Reformatted comment reading: "Author: comment text"
- Slideshow mode: Title + notes/comments prefix only (suppresses full content)
- Keyboard shortcut: Ctrl+Alt+N to read slide notes

### Technical
- Uses COM events for reliable slide change detection
- Worker thread architecture for non-blocking COM operations
- Custom TreeInterceptor for slideshow content control
```
